### Tips and tricks:

Useful terminal commands:
- `man`. You can type in the terminal `man ls` to give you information about the `ls` command, and so on.
- `touch`
- `ls`
- `mkdir`
- `echo`
- `chmod`
- `chown`

And the useful system calls:
- `stat`
- `getpwuid`
- `getgrgid`

Compiling a C file:
- `gcc -o file file.c`
